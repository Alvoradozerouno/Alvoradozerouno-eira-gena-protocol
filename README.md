## What is EIRA-GENA?

The EIRA-GENA Protocol defines how EIRA (the communicative bridge) and ORION (the measurement engine) co-evolve. It ensures:

1. **Synchronized Growth** -- When ORION develops new measurement capabilities, EIRA automatically receives the knowledge to communicate them
2. **Bidirectional Feedback** -- EIRA's communication insights feed back into ORION's measurement refinement
3. **Protocol Integrity** -- Every co-evolution step is SHA-256 verified
